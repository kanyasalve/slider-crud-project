# Slider CRUD Project

A dynamic Slider Management System built using PHP, MySQL, HTML5, CSS3, Bootstrap, and jQuery.

This project allows admin users to create, read, update, and delete slider content through an admin panel and display it dynamically on the frontend with tab-based sliders (Desktop) and accordion-based sliders (Mobile).

## üöÄ Features

### Admin Panel
- Add new slider
- Edit slider
- Delete slider
- Upload slide images
- Manage tab titles and slide content

### Frontend (Desktop View)
- Left Column: Tabs
- Middle Column: Slider (content)
- Right Column: 1:1 image display
- Slide change updates image automatically
- Controls attached only to content slider

### Frontend (Mobile View)
- Tabs convert into Accordion
- Slider uses images as background
- Fully responsive layout
